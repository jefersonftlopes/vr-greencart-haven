Micro frontend - Module Federation
Instruções
Desenvolva uma aplicação em React JS que seja um micro front-end utilizando
Module Federation.
A aplicação deve conter um app central e mais 3 aplicações sendo elas Header,
Footer, Cards que deve ter uma lista de produtos. a aplicação deve ter
uma interação da lista de produtos com o Header onde o mesmo ira exibir um
modal com a lista de produtos selecionada.
Para o desenvolvimento dessa aplicação você poderá utilizar as APIs abaixo:
Layout: https://www.figma.com/community/file/1322143172205405825/greencart-haven-website-landing-page-design
Produtos: https://dummyjson.com/docs/products
Cart: https://dummyjson.com/docs/carts
Você é livre para definir as demais bibliotecas que irá utilizar no projeto.
Seu projeto deve possuir os seguintes scripts npm:
start: inicia o projeto para desenvolvimento
build: gera os artefatos para deploy em produção
